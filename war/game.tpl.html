Push Board

A web application using Comet (aka HTTP Server Push) Technology Stack: HTML 5, JavaScript jQuery, Java EE, Java Persistence API (JPA).

Please press the "Connect" button to communicate to other players. Below the board, you will see a number assigned to you. After the game start, you must press on your assigned number as it appears on the board until the game is over. The first player that reaches 30 points wins.

Your browser doesn't support canvas.
